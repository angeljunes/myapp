# Assets - Images

Este directorio debe contener las imágenes de la aplicación RCAS.

## Imágenes requeridas:
- `logo.png` - Logo principal de la aplicación RCAS
- `splash_background.png` - Imagen de fondo para la pantalla de splash
- `map_marker_alta.png` - Marcador personalizado para alertas de prioridad ALTA
- `map_marker_media.png` - Marcador personalizado para alertas de prioridad MEDIA  
- `map_marker_baja.png` - Marcador personalizado para alertas de prioridad BAJA
- `emergency_button.png` - Icono para el botón de emergencia
- `user_placeholder.png` - Imagen placeholder para usuarios sin foto de perfil

## Formatos recomendados:
- PNG con transparencia para iconos
- JPG para imágenes de fondo
- Resoluciones: 1x, 2x, 3x para diferentes densidades de pantalla

## Nota:
Actualmente la aplicación funciona sin estas imágenes usando iconos de Material Design por defecto.
