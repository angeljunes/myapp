# Assets - Icons

Este directorio debe contener los iconos personalizados de la aplicación RCAS.

## Iconos requeridos:
- `rcas_logo.svg` - Logo vectorial de RCAS para diferentes tamaños
- `alert_high.svg` - Icono para alertas de prioridad alta
- `alert_medium.svg` - Icono para alertas de prioridad media
- `alert_low.svg` - Icono para alertas de prioridad baja
- `emergency.svg` - Icono del botón de emergencia
- `location.svg` - Icono de ubicación personalizado
- `user.svg` - Icono de usuario personalizado
- `settings.svg` - Icono de configuración

## Formatos recomendados:
- SVG para iconos vectoriales (escalables)
- PNG en múltiples resoluciones (24dp, 48dp, 72dp) si se requiere rasterización

## Colores:
- Usar colores consistentes con el tema de la app
- Prioridad ALTA: #F44336 (rojo)
- Prioridad MEDIA: #FF9800 (naranja)  
- Prioridad BAJA: #4CAF50 (verde)
- Primario: #000000 (negro)

## Nota:
Actualmente la aplicación funciona sin estos iconos usando Material Design Icons por defecto.
